区分变量、属性、函数、方法

  • 变量：单独声明、单独存在
  • 属性：对象里面的变量称为属性，不需要声明，是用来描述对象的特征

相同点：都是用来存储数据的

// 变量
var num = 10;
console.log(num)

// 属性
var people = {
    age: 10
}
console.log(people.age)
  • 函数：单独存在的，通过“函数名()”的方式就可以调用
  • 方法：对象里面的函数称为方法，方法不需要声明，使用“对象.方法名()”的方式可以调用，方法用来描述对象的行为和功能

相同点：都是实现某种功能

// 函数
function fn() {
    console.log("yes!")
}
fn()

// 方法
var people = {
    fn: function() {
        console.log("yes!")
    }
}
people.fn()
